Capabilities

Built to do, not just to chat

Nebula combines long-horizon reasoning with native tool use, so it can carry a task from intent to outcome.

Long-horizon planning

Breaks fuzzy goals into ordered subtasks, tracks its own progress, and recovers gracefully when a step fails.

Native tool use

Calls APIs, runs code, queries databases, and browses the web — all within a single, governed execution loop.

2M token context

Hold an entire codebase, deal room, or research corpus in working memory without lossy chunking or retrieval gaps.

Guardrails built in

Policy-aware reasoning with audit trails, content filtering, and configurable approval steps for sensitive actions.

Streaming latency

First token in under 120ms and sustained high throughput, tuned for interactive agents and live experiences.

Multimodal in & out

Reason over text, images, audio, and structured data in a single prompt, then return whichever format fits the job.
